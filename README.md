# TMBD API

## Pré-requisitos

- Node.js 16+ e npm ou yarn

## Como usar (desenvolvimento)

1. Instale dependências:

```bash
npm install
# ou
yarn
```

2. Rode o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
```

3. Abra http://localhost:5173 (ou a porta indicada no terminal).
